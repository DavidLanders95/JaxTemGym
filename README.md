# Jax Tem Gym

A differentiable version of TemGym: Landers, D., Clancy, I., Weber, D., Dunin-Borkowski, R. & Stewart, A. (2023). J. Appl. Cryst. 56, https://doi.org/10.1107/S1600576723005174
used here to seamlessly determine linear transformation matrices of optical system, in order to calibrate the coordinate system of a 4D stem experiment.
